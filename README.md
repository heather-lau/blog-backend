# Blog Backend

## Getting Started
### Prerequisites

Before starting make sure your computer has the following installed:

* [npm](https://www.npmjs.com/get-npm/)
* [Node.js](https://nodejs.org/)
* [MongoDB](https://www.mongodb.com/)

Start the database:
```
mongod
```

Running the app:
```
npm start
```
Or
```
npm run dev
```
